# 補正計算ページ作成タスク

## 目的
5体分の「基準値 + 補正率%」を入力し、各体の補正後の結果値と増加分を計算、結果値の大きい順に順位を付けて表示するWebページを作成する。

## 仕様
- フォルダ: `補正計算/index.html`
- 入力: 5体分（各体ごとに「名前(任意)」「基準値」「補正1%」「補正2%」「補正3%」）
- 計算（加算方式）:
  - 合計補正率 = 補正1 + 補正2 + 補正3
  - 増加分 = 基準値 × (合計補正率/100)
  - 結果値 = 基準値 + 増加分 = 基準値 × (1 + 合計補正率/100)
- 表示: 結果値の大きい順に順位（①②③④⑤）を付与、補正内訳（a+b+c=合計%）も併記
- リアルタイム計算（入力変更で即更新）
- スタイル: 既存サイト（index.html）と統一感のあるシアン/ブルー系ダークテーマ
- レスポンシブ対応（スマホ縦並び）

## レビュー
### 実装内容
1. **`補正計算/index.html` 新規作成**
   - 1ファイル完結（HTML + 内部CSS + 内部JS、外部依存はGoogle Fontsのみ）
   - 既存トップページと同系統のシアン/ブルー系グラスモーフィズム
   - 5行の入力（名前・基準値・補正率%）→ リアルタイム計算 → 大きい順カード表示
   - 順位は①〜⑤、1位/2位/3位で枠の左ボーダー色を金・銀・銅に変化
